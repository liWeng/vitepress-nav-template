#### mk-left-right

#### 应用场景

* 左边固定宽度, 右边自动填充满
    
#### 用法

``` html
<mk-left-right  left_width="400px" :height="testheight">
            
            <mk-panel slot="left" height="100%">
    
            
            </mk-panel>
            
            <mk-panel slot="right" height="100%">
                
            </mk-panel>
            
</mk-left-right>

```
