## Summer-Vue
  fweb对应的前端组件框架，基于vue1.0版本开发
  基于jquery、bootstrap、vuejs等基础库，组件一个具有mvvm特性、组件式开发的HTML UI库

## 时间：2016

## 项目使用
    服务于山西能投、山西地电、四川能投、四川水电所有项目

## 基本使用
*使用前确保本地安装了node环境，建议版本号大于6.0*

### 初始化环境依赖包，以下命令在工程目录执行
``` npm install ```

### 编译
``` npm run build ```

### 运行
``` npm run dev ```
